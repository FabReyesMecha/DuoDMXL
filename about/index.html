About this Library

This site provides documentation for the library DuoDMXL intended for communicating a Duo (RedBear) microcontroller board with DYNAMIXEL servomotors. The library was designed with servos MX-64AR as its main target so it may need some modifications for other models.

Additionally, schematics and board designs are provided for faster prototyping.
